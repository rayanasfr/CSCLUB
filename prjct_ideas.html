•Learn more about python by discovering new libraries and modules

• Flask module, a microframework!

Flask is a popular micro web framework for building web applications using the Python programming language. It's known for its simplicity, flexibility, and ease of use. Learn how to use Flask by developing your own portfolio. Check out this amazing video to help you achieve it

•Tkinter library, a graphical user interface!

Tkinter is indeed a standard GUI library for Python that enables developers to create interactive and visually appealing desktop applications. It comes with a variety of widgets and components that can be used to design the user interface of your application. These components include windows, buttons, labels, text boxes, canvas, and more. Tkinter is especially popular for its simplicity and ease of use, making it a great choice for beginners who want to start developing desktop applications. Discover this library by building an age calculator app and have fun personalizing it.
